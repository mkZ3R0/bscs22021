Kitabcha A Manga Library Management And Reading APP


This is a group project in our university's Database course. Me along with 3 other amazing teammates developed a mobile application in which users can have multiple profiles, and manage their own library of mangas by making custom categories. The users can read mangas through the app and the app automatically keeps track of their progress. The application was built in KOTLIN, where we used JET-PACK Compose for the UX and ROOM DATABASE for implementing the backend database.
