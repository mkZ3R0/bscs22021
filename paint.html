A Paint Application in Python


This was a univeristy project in my object oriented programming course. I developed a paint application using TKINTER Library in python. The application supports all basic paint functionality, allowing different shapes, brushes, save, load, and erase features. The canvas also includes zoom, select and move functionality for editing.
